## 🎮 Final Project

### 🐱 Cat Night Walk (Adventure Game)

A multi-level interactive game built with Python and Pygame.

- Navigate through a maze while avoiding enemies  
- Cross a road while dodging falling obstacles  
- Complete a stealth mission by collecting items  
- Includes win/loss screens and a final ending
